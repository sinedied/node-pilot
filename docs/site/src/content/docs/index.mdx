**Cockpit.js** is a [Copilot canvas extension](https://docs.github.com/en/copilot/how-tos/github-copilot-app/working-with-canvas-extensions)
that puts your project's whole inner loop in the Copilot side panel. It is
zero-config: it auto-detects your package manager, scripts, framework, test
runner, linter, formatter and TypeScript, and degrades gracefully when a
capability is missing.
